# 📘 Chapitre 16 — Héritage et Polymorphisme
Python — Du zéro à l’expert

## 🎯 Objectif

Approfondir la **programmation orientée objet (POO)** en Python en explorant les concepts d’héritage et de polymorphisme.

Ces mécanismes permettent de créer des structures de code plus flexibles, réutilisables et extensibles.

Dans ce chapitre, nous explorons :

- L’héritage simple entre classes
- La fonction `super()` pour appeler la classe parente
- Le polymorphisme
- Les classes abstraites
- L’héritage multiple
- Les propriétés avec `@property`
- Les méthodes `@staticmethod` et `@classmethod`

## 🧠 Compétences acquises

- Comprendre l’héritage entre classes
- Utiliser le polymorphisme pour rendre le code plus flexible
