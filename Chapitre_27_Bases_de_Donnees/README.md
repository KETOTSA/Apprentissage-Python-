# 📘 Chapitre 27 — Bases de Données avec Python
Python — Du zéro à l’expert

## 🎯 Objectif

Comprendre comment stocker et manipuler des données avec des bases de données en Python.

Ce chapitre introduit les concepts essentiels pour gérer des données de manière structurée et persistante.

Dans ce chapitre, nous explorons :

- SQLite — La base de données embarquée
- CRUD — Créer, Lire, Mettre à jour, Supprimer
- SQL avancé — Jointures et agrégations
- SQLAlchemy — L’ORM Python
- Gestion des transactions

## 📂 Fichiers

- `python_chapitre_27.pdf`

## 🧠 Compétences acquises

- Manipuler une base de données avec SQLite
- Effectuer des opérations CRUD
- Écrire des requêtes SQL avancées
- Utiliser SQLAlchemy pour interagir avec une base de données
- Gérer les transactions de manière sécurisée
