# 📘 Chapitre 09 — Les Tuples  
Python — Du zéro à l’expert

## 🎯 Objectif

Découvrir les **tuples**, une structure de données proche des listes mais **immuable**, utile pour stocker des informations qui ne doivent pas être modifiées.

Dans ce chapitre, nous explorons :

- Créer un tuple
- Accéder aux éléments
- L’immuabilité : force et limites
- Le déballage (unpacking)
- Les tuples et les fonctions
- Tuple vs Liste — Quand choisir ?
- Les namedtuples
- Les tuples comme clés de dictionnaire

## 🧠 Compétences acquises

- Comprendre l’immuabilité des tuples
- Manipuler et accéder aux éléments
- Utiliser l’unpacking pour écrire un code plus propre
- Choisir entre tuples et listes selon le contexte
