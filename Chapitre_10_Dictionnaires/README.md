# 📘 Chapitre 10 — Les Dictionnaires  
Python — Du zéro à l’expert

## 🎯 Objectif

Découvrir les **dictionnaires**, une structure de données essentielle en Python basée sur le principe **clé-valeur**.

Les dictionnaires permettent d'organiser et d'accéder rapidement à des informations.

Dans ce chapitre, nous explorons :

- Créer un dictionnaire
- Accéder aux valeurs
- Modifier un dictionnaire
- Parcourir un dictionnaire
- Les dictionnaires imbriqués
- La dictionary comprehension
- Les méthodes utiles des dictionnaires
- Cas pratiques : comptage et regroupement

## 🧠 Compétences acquises

- Stocker et organiser des données avec des clés
- Manipuler et modifier des dictionnaires
- Parcourir efficacement les données
- Utiliser les compréhensions de dictionnaires pour écrire un code plus propre
