# 📘 Chapitre 13 — Fonctions Avancées
Python — Du zéro à l’expert

## 🎯 Objectif

Explorer les **fonctions avancées en Python** afin d’écrire un code plus flexible, puissant et expressif.

Ces concepts permettent de manipuler les fonctions de manière plus avancée et d'utiliser des techniques de programmation fonctionnelle.

Dans ce chapitre, nous explorons :

- *args — arguments positionnels variables
- **kwargs — arguments nommés variables
- Les fonctions lambda
- map() — transformer une séquence
- filter() — filtrer une séquence
- reduce() — réduire une séquence
- Les closures
- Les décorateurs

- Écrire des fonctions plus flexibles
- Utiliser des fonctions anonymes
