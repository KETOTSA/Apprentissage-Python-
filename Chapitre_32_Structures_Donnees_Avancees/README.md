# 📘 Chapitre 32 — Structures de Données Avancées
Python — Du zéro à l’expert

## 🎯 Objectif

Découvrir et maîtriser les structures de données avancées de Python pour améliorer l’efficacité et les performances des programmes.

Ces structures sont largement utilisées dans les algorithmes, le traitement de données et les systèmes performants.

Dans ce chapitre, nous explorons :

- `defaultdict` — Dictionnaire avec valeur par défaut
- `Counter` — Compter les occurrences
- `deque` — File double à deux bouts
- `heapq` — File de priorité
- `bisect` — Recherche et insertion binaire
- `OrderedDict` et `ChainMap`
