# 📘 Chapitre 23 — NumPy et Pandas
Python — Du zéro à l’expert

## 🎯 Objectif

Découvrir les bibliothèques fondamentales de la Data Science en Python : NumPy et Pandas.

Ce chapitre permet de manipuler efficacement des données, de réaliser des analyses et de préparer des datasets pour le Machine Learning.

Dans ce chapitre, nous explorons :

- NumPy — Les tableaux multidimensionnels
- NumPy — Opérations vectorisées
- NumPy — Indexation et slicing avancés
- Pandas — Series et DataFrame
- Pandas — Filtrage et transformation
- Pandas — Agrégations et groupby
- Pandas — Gestion des données manquantes
- Pandas — Fusion et jointure de données
