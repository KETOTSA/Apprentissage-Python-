# 📘 Chapitre 20 — Itérateurs et Générateurs
Python — Du zéro à l’expert

## 🎯 Objectif

Comprendre le fonctionnement des itérateurs et des générateurs pour produire des données de manière efficace et optimisée.

Ces concepts permettent de manipuler de grandes quantités de données sans surcharger la mémoire.

Dans ce chapitre, nous explorons :

- Le protocole itérateur
- Créer son propre itérateur
- Les générateurs avec `yield`
- Les expressions génératrices
- Générateurs avec plusieurs yield
- `yield from` pour déléguer à un sous-générateur
- Le module `itertools`
- Générateurs avancés : send() et throw()
