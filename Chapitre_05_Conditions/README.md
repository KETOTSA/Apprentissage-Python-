# 📘 Chapitre 05 – Les Conditions  
Python — Du zéro à l’expert

## 🎯 Objectif

Apprendre à Python à prendre des décisions grâce aux structures conditionnelles :

- if : la décision de base
- if / else : deux chemins possibles
- if / elif / else : plusieurs conditions
- Conditions imbriquées
- Opérateurs logiques (and / or / not)
- Opérateur ternaire
- Valeurs truthy et falsy
- match / case (Python 3.10)
