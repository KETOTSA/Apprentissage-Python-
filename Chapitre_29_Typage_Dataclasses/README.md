# 📘 Chapitre 29 — Typage Statique et Dataclasses
Python — Du zéro à l’expert

## 🎯 Objectif

Améliorer la qualité du code Python grâce au typage statique et simplifier la création de classes avec les dataclasses.

Dans ce chapitre, nous explorons :

- Les annotations de type — Type hints
- Le module typing — Types avancés
- mypy — Vérification statique des types
- dataclasses — Classes de données simplifiées
- Dataclasses avancées — frozen, order, slots
- Generic et Protocol — Typage avancé
