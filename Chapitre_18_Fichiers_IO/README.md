# 📘 Chapitre 18 — Fichiers et I/O
Python — Du zéro à l’expert

## 🎯 Objectif

Apprendre à manipuler les fichiers en Python afin de lire, écrire et organiser des données dans le système de fichiers.

Ces compétences sont essentielles pour travailler avec des données réelles et construire des applications capables de persister des informations.

Dans ce chapitre, nous explorons :

- Ouvrir et fermer un fichier
- Lire un fichier
- Écrire dans un fichier
- Manipuler les fichiers au format JSON
- Manipuler les fichiers au format CSV
- Naviguer dans les dossiers avec `pathlib`
- Cas pratique : création d’un gestionnaire de notes

## 📂 Fichiers

- `Python_chapitre_18.pdf`

## 🧠 Compétences acquises

- Lire et écrire des fichiers en Python
- Manipuler les formats de données JSON et CSV
- Naviguer dans le système de fichiers
- Créer de petits outils de gestion de données
