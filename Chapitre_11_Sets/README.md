# 📘 Chapitre 11 — Les Ensembles (Sets)
Python — Du zéro à l’expert

## 🎯 Objectif

Découvrir les **ensembles (sets)** en Python, une structure de données qui permet de stocker des éléments **uniques** et d’effectuer des **opérations mathématiques sur les ensembles**.

Dans ce chapitre, nous explorons :

- Créer un ensemble
- Les propriétés fondamentales des sets
- Ajouter et supprimer des éléments
- Les opérations ensemblistes
- Les relations entre ensembles
- La set comprehension
- Le frozenset (ensemble immuable)
- Cas pratiques : dédoublonnage et comparaisons

## 🧠 Compétences acquises

- Comprendre l’unicité des éléments dans un set
- Utiliser les opérations ensemblistes (union, intersection, différence)
- Manipuler efficacement des collections sans doublons
- Utiliser les sets pour comparer des données
