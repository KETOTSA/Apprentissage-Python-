# 📘 Chapitre 12 — Les Fonctions de Base
Python — Du zéro à l’expert

## 🎯 Objectif

Comprendre les **fonctions**, un concept fondamental en programmation qui permet d'organiser, réutiliser et structurer le code.

Les fonctions permettent de diviser un programme en blocs logiques et réutilisables.

Dans ce chapitre, nous explorons :

- Définir et appeler une fonction
- Les paramètres et arguments
- La valeur de retour (return)
- Les valeurs par défaut
- La portée des variables (scope)
- Retourner plusieurs valeurs
- Les fonctions comme objets
- Documenter ses fonctions

- Structurer un programme avec des fonctions
- Comprendre la portée des variables
