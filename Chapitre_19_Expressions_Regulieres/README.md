# 📘 Chapitre 19 — Expressions Régulières
Python — Du zéro à l’expert

## 🎯 Objectif

Comprendre et utiliser les expressions régulières pour rechercher, valider et manipuler du texte efficacement.

Les regex sont un outil puissant pour traiter des données textuelles, très utilisé en programmation et en data.

Dans ce chapitre, nous explorons :

- Les métacaractères fondamentaux
- Les fonctions du module `re`
- re.findall() et re.finditer()
- Les groupes de capture
- re.sub() pour remplacer du texte
- re.split() pour découper une chaîne
- Compiler un pattern avec re.compile()
- Cas pratiques : validations courantes
