# 📘 Chapitre 08 — Les Listes  
Python — Du zéro à l’expert

## 🎯 Objectif

Découvrir les **listes**, l'une des structures de données les plus utilisées en Python pour stocker et manipuler plusieurs éléments.

Dans ce chapitre, nous explorons :

- Créer une liste
- Accéder aux éléments (indexation et slicing)
- Ajouter des éléments
- Supprimer des éléments
- Rechercher dans une liste
- Trier une liste
- Copier une liste
- Les listes imbriquées (matrices)
- La list comprehension
