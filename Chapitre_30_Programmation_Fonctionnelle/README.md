# 📘 Chapitre 30 — Programmation Fonctionnelle
Python — Du zéro à l’expert

## 🎯 Objectif

Découvrir les principes de la programmation fonctionnelle pour écrire un code plus clair, modulaire et maintenable.

Dans ce chapitre, nous explorons :

- Fonctions pures et immutabilité
- functools — La boîte à outils fonctionnelle
- operator — Fonctions pour les opérateurs
- Composition de fonctions
- Currying et application partielle avancée
- Monades légères — Maybe et Result
