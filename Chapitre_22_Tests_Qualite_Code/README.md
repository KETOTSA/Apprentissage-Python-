# 📘 Chapitre 22 — Tests et Qualité du Code
Python — Du zéro à l’expert

## 🎯 Objectif

Comprendre l’importance des tests en programmation et apprendre à écrire du code fiable, maintenable et professionnel.

Ce chapitre introduit les outils et bonnes pratiques pour garantir la qualité du code.

Dans ce chapitre, nous explorons :

- Pourquoi tester ?
- unittest — Le framework standard
- Les assertions de unittest
- pytest — Le framework moderne
- Les mocks — Simuler les dépendances
- Organisation des tests et couverture
- Le TDD (Test Driven Development)

## 📂 Fichiers

- `Python_chapitre_22.pdf`

## 🧠 Compétences acquises

- Comprendre l’importance des tests logiciels
- Écrire des tests unitaires avec unittest
- Utiliser pytest pour des tests modernes
- Simuler des comportements avec des mocks
- Mettre en place une stratégie de tests efficace
