# 📘 Chapitre 15 — Introduction à la Programmation Orientée Objet (POO)
Python — Du zéro à l’expert

## 🎯 Objectif

Découvrir les bases de la **programmation orientée objet (POO)** en Python, une approche qui permet de modéliser des objets du monde réel à l’aide de classes et d’instances.

Dans ce chapitre, nous explorons :

- Les concepts fondamentaux de la POO
- Créer une classe avec `class`
- Le constructeur `__init__` et `self`
- Les attributs de classe et attributs d’instance
- Les méthodes spéciales (dunder methods)
- Un exemple complet : classe Compte Bancaire
- Les fonctions `isinstance()` et `type()`
