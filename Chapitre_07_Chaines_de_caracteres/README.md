# 📘 Chapitre 07 – Les Chaînes de Caractères  
Python — Du zéro à l’expert

## 🎯 Objectif

Comprendre comment manipuler le texte en Python grâce aux chaînes de caractères (strings).

Dans ce chapitre, nous explorons :

- Créer une chaîne de caractères
- Accéder aux caractères (indexation)
- Le slicing pour découper une chaîne
- L’immuabilité des chaînes
- Les méthodes de transformation
- Diviser et assembler des chaînes
- Vérifier le contenu d’une chaîne
- Aligner et remplir des chaînes
- Les chaînes multilignes
