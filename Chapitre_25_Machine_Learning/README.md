# 📘 Chapitre 25 — Machine Learning avec Scikit-learn
Python — Du zéro à l’expert

## 🎯 Objectif

Découvrir les bases du Machine Learning avec Scikit-learn et comprendre comment construire, entraîner et évaluer des modèles prédictifs.

Ce chapitre introduit les concepts fondamentaux utilisés en Data Science.

Dans ce chapitre, nous explorons :

- Le workflow du Machine Learning
- Régression — Prédire des valeurs continues
- Classification — Prédire une catégorie
- Clustering — Regrouper sans étiquettes
- Pipeline — Enchaîner les étapes

## 🧠 Compétences acquises

- Comprendre le cycle de vie d’un modèle de Machine Learning
- Construire des modèles de régression et de classification
- Appliquer des techniques de clustering
- Structurer un pipeline de traitement et de modélisation
