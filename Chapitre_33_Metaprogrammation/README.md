# 📘 Chapitre 33 — Métaprogrammation
Python — Du zéro à l’expert

## 🎯 Objectif

Comprendre comment Python permet de manipuler et générer du code dynamiquement.

La métaprogrammation permet d’écrire du code qui agit sur du code, un concept avancé utilisé dans les frameworks et bibliothèques professionnelles.

Dans ce chapitre, nous explorons :

- Introspection — Explorer les objets Python
- Descripteurs — Contrôler l’accès aux attributs
- Métaclasses — Fabriquer des classes dynamiquement
- `__init_subclass__` — Alternative moderne aux métaclasses
- `contextlib` — Gestionnaires de contexte avancés
- Décorateurs de classe et `@class_getitem`
