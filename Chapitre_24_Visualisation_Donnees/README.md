# 📘 Chapitre 24 — Visualisation de Données
Python — Du zéro à l’expert

## 🎯 Objectif

Apprendre à représenter visuellement les données afin de mieux les comprendre, les analyser et communiquer des résultats clairs.

La visualisation est une compétence essentielle en Data Science.

Dans ce chapitre, nous explorons :

- L’écosystème de visualisation Python
- Matplotlib — Les bases
- Seaborn — Visualisation statistique
- Plotly — Graphiques interactifs
- Bonnes pratiques de visualisation

## 📂 Fichiers

- `Python_chapitre_24.pdf` 

## 🧠 Compétences acquises

- Créer des graphiques avec Matplotlib
- Réaliser des visualisations statistiques avec Seaborn
- Construire des graphiques interactifs avec Plotly
- Améliorer la lisibilité et l’impact des graphiques
