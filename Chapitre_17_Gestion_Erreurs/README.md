# 📘 Chapitre 17 — Gestion des Erreurs
Python — Du zéro à l’expert

## 🎯 Objectif

Comprendre comment gérer efficacement les erreurs en Python afin d'écrire des programmes plus robustes et fiables.

La gestion des exceptions permet d’anticiper les situations imprévues et d’éviter l’arrêt brutal d’un programme.

Dans ce chapitre, nous explorons :

- Les types d’erreurs en Python
- La hiérarchie des exceptions
- try / except pour intercepter les erreurs
- else et finally pour compléter la structure
- Intercepter plusieurs exceptions
- raise pour lever ses propres exceptions
- Créer des exceptions personnalisées
- Le gestionnaire de contexte avec `with`
- Les assertions pour vérifier les invariants
