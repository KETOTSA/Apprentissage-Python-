# 📘 Chapitre 26 — APIs et Web avec Python
Python — Du zéro à l’expert

## 🎯 Objectif

Comprendre comment interagir avec des APIs et développer des services web en Python.

Ce chapitre permet de connecter Python au monde réel via des échanges de données.

Dans ce chapitre, nous explorons :

- Le protocole HTTP — Les bases
- requests — Consommer une API
- Flask — Créer une API REST
- FastAPI — L’API moderne
- Web scraping — Extraire des données du web

## 🧠 Compétences acquises

- Comprendre le fonctionnement des requêtes HTTP
- Consommer des APIs externes
- Créer une API REST avec Flask
- Développer des APIs performantes avec FastAPI
- Extraire des données depuis le web
