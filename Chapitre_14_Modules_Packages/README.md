# 📘 Chapitre 14 — Les Modules et Packages
Python — Du zéro à l’expert

## 🎯 Objectif

Découvrir comment **organiser et réutiliser du code en Python** grâce aux modules et aux packages.

Les modules permettent d'importer des fonctionnalités existantes, tandis que les packages permettent de structurer des projets plus complexes.

Dans ce chapitre, nous explorons :

- Importer un module avec `import`
- Le module math — calculs mathématiques
- Le module random — génération aléatoire
- Le module os — interaction avec le système
- Le module sys — environnement Python
- Le module datetime — gestion des dates et heures
- Créer ses propres modules
- La variable spéciale `__name__`
- Les packages pour organiser les modules
