# 📘 Chapitre 21 — Concurrence et Parallélisme
Python — Du zéro à l’expert

## 🎯 Objectif

Comprendre comment exécuter plusieurs tâches simultanément en Python afin d'améliorer les performances et l'efficacité des programmes.

Ce chapitre introduit les différentes approches de concurrence et de parallélisme.

Dans ce chapitre, nous explorons :

- Concurrence vs Parallélisme
- threading — Exécution concurrente
- Synchronisation avec Lock et Event
- ThreadPoolExecutor — Gestion des threads
- multiprocessing — Parallélisme réel
- asyncio — Programmation asynchrone
- Choisir la bonne approche selon le contexte
