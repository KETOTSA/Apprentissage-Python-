# 📘 Chapitre 06 – Les Boucles  
Python — Du zéro à l’expert

## 🎯 Objectif

Comprendre comment automatiser la répétition en Python grâce aux boucles :

- La boucle while — répéter tant qu’une condition est vraie
- La boucle for — parcourir une séquence
- La fonction range()
- break — sortir d’une boucle
- continue — passer à l’itération suivante
- pass — instruction vide
- La clause else dans les boucles
- Les boucles imbriquées
- enumerate() et zip()
